# Local RAG Application

Microsoft Foundry Local kullanarak çevrimdışı çalışan yerel bir Soru-Cevap asistanı.

## Proje Yapısı
- `data/` - Bilgi tabanı belgeleri
- `src/` - Kaynak kodlar
- `notebooks/` - Deney notebook'ları

## Kurulum
```bash
pip install -r requirements.txt
```

## Kullanım
```bash
python main.py
```
